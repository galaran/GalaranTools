RandomLoot
==========

Плагин предназначен для выдачи предметов игрокам (определенных или случайных из списка или таблицы)

* Набор (Kit) содержит стаки предметов  
* Список наборов (Kit List) содержит несколько наборов (возможны повторы). При выдаче, выбирается один случайный  
* Таблица Лута (Loot Table) содержит несколько списков, каждый имеет вес. При выдаче, вначале случайно по весу выбирается список, далее из списка один случайный набор

При выдаче (give) можно задавать количество повторений, указав `*<количество>` после названия  
Каждое повторение роллится отдельно. Все непоместившиеся предметы будут дропнуты поблизости от игрока  
Также наборы можно бесконечно получать кликом по специальным табличкам  

Наборы
------

`/rloot kit add <kit_name>` - сохранить содержимое инвентаря, как набор (или заменить)  
`/rloot kit list` - список наборов  
`/rloot kit remove <kit_name>` - удалить набор  
`/rloot kit info <kit_name>` - посмотреть содержимое набора  
`/rloot kit give <kit_name>[*amount] <player>` - выдать набор игроку amount раз  


Можно поставить табличку, при использовании которой игроки смогут бесконечно получать набор:

    [               ]
    [     [kit]     ]
    [     <kit>     ]
    [               ]

Непоместившиеся предметы будут проигнорированы  
Возможно ограничить использование таких табличек только в некоторых мирах

Списки наборов
--------------

`/rloot list add <list_name>` - создать список  
`/rloot list list` - посмотреть все списки наборов  
`/rloot list remove <list_name>` - удалить список  
`/rloot list info <list_name>` - посмотреть содержимое списка  
`/rloot list give <list_name>[*amount] <player>` - выдать случайный набор из списка amount раз

`/rloot list addkit <list_name> <kit_name>` - добавить набор в список (могут повторяться)  
`/rloot list rmkit <list_name> <kit_name>` - удалить набор из списка (первый, если их несколько)

Таблицы лута
------------

`/rloot table add <table_name>` - создать таблицу  
`/rloot table list` - список таблиц  
`/rloot table remove <table_name>` - удалить таблицу  
`/rloot table info <table_name>` - посмотреть содержимое таблицы  
`/rloot table give <table_name>[*amount] <player>` - выдать игроку набор из таблицы amount раз

`/rloot table addlist <table_name> <list_name> <weight>` - добавить (или заменить) список в таблицу с весом  
`/rloot table rmlist <table_name> <list_name>` - удалить список из таблицы  

При выдаче вначале выбирается список по весу. Как это работает:  
Например, есть списки:

    100 - Обычное: [(16 яблок, 8 хлеба) (32 дерева)]
    20 - Редкое: [(Меч)]
    5 - Легендарное: [(Кирка Silk Touch) (Кирка Fortune)]
    
_() - набор, [] список_

На каждые 100 выпадений обычного лута, всреднем будет выпадать 20 редкого и 5 легендарного  
Если добавить ещё одну категорию (список) с весом 5000, то шанс остальных резко упадет  
После того как выбран список, из него случайно выбирается один набор

Конфигурация
------------

    debug: true # Вывод сообщений отладки
    lang: english # Какой .lang файл использовать для сообщений

    kit-signs:
        restrict: true # Ограничивать использование табличек наборов
        allowed-worlds: [Aertos_pvp] # Миры, в которых разрешено использовать таблички наборов

`/rloot reload` - перегрузить главный конфиг, тексты и наборы + списки + таблицы

Пермишены
---------

`randomloot.command` - доступ ко всем командам (op)  
`randomloot.kitsign.create` - создание табличек наборов (op)  
`randomloot.kitsign.bypassworldcheck` - игнорирование ограничения на использование табличек наборов в мирах (false)
